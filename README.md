# Nexus AI — AI Dashboard Template

![Next.js](https://img.shields.io/badge/Next.js-16-black)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-38bdf8)
![License](https://img.shields.io/badge/License-MIT-green)

A production-ready AI dashboard template built with 
Next.js 16 and TypeScript. Dark luxury design with 
light mode included.

## Live Demo
> nexus-ai-landing-lilac.vercel.app

## Pages Included
- `/` — Landing page (Hero, Features, Pricing, FAQ)
- `/login` — Login page with validation
- `/register` — Register page with validation
- `/blog` — Blog list page
- `/blog/[slug]` — Blog detail page
- `/docs` — Documentation page with sidebar
- `/dashboard` — Full dashboard UI preview

## Features
- Dark/Light mode toggle
- Smooth Framer Motion animations
- Loading screen
- SweetAlert2 popups
- Syntax highlighting (Shiki)
- Fully responsive (mobile-first)
- Form validation
- Dashboard with 5 tabs
- CSS variables design system

## Tech Stack
- Next.js 16 + TypeScript
- Tailwind CSS
- Framer Motion
- next-themes
- SweetAlert2
- Shiki
- Lucide React

## Getting Started

1. Clone or download the repository
2. Install dependencies:
   npm install
3. Run development server:
   npm run dev
4. Open http://localhost:3000

## Build for Production
npm run build
npm run start

## Design System
- Background: #131313 (dark) / #F8F6F1 (light)
- Accent: #E8D5B7 (gold)
- Fonts: Noto Serif + Manrope + Inter
- Fully CSS variables based
