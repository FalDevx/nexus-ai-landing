# Nexus AI — SaaS Landing Page Template

## Tech Stack
- Next.js 16
- Tailwind CSS
- Framer Motion
- TypeScript
- next-themes

## Pages Included
- / Landing page (Hero, Features, Pricing, FAQ, etc)
- /login Login page
- /register Register page  
- /blog Blog list page
- /blog/[slug] Blog detail page
- /docs Documentation page
- /dashboard Dashboard preview

## Getting Started
1. npm install
2. npm run dev
3. Open http://localhost:3000

## Design System
- Dark luxury minimal theme
- Light mode included
- Fully responsive
- CSS variables based
